# 🐦 Mini-Twitter

Mini-Twitter é uma aplicação web simples inspirada no Twitter. Permite que os usuários se cadastrem, façam login, publiquem tweets de até 280 caracteres, curtam posts e editem suas informações de perfil. Todos os dados são salvos localmente no navegador via `localStorage`.

## 🚀 Funcionalidades

- Cadastro e login de usuários
- Manter usuário logado após refresh
- Postagem de tweets com até 280 caracteres
- Curtir tweets
- Listagem de tweets do usuário logado
- Edição de perfil
- Layout responsivo (Mobile First)
- Armazenamento local via `localStorage`

## 🛠️ Tecnologias Utilizadas

- HTML5 semântico
- CSS3 com Flexbox e responsividade
- JavaScript moderno (ES6+)
- Armazenamento via `localStorage`
